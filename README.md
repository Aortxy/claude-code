## 📖 Overview
Repositori ini berisi struktur kode sumber dari **Claude Code**, sebuah alat CLI (Command Line Interface) yang dikembangkan oleh Anthropic. Proyek ini dibangun menggunakan arsitektur modern berbasis Node.js dengan integrasi komponen UI terminal yang canggih.

## 🗂️ Project Structure
Berikut adalah direktori utama yang terdapat dalam proyek ini:

| FOLDER | DESKRIPSI |
| :--- | :--- |
| `assistant` | Logika utama asisten AI |
| `bootstrap` | Inisialisasi dan konfigurasi awal |
| `cli` | Entry point untuk Command Line Interface |
| `commands` | Daftar perintah yang tersedia di terminal |
| `components` | Komponen UI (React/Ink) untuk tampilan terminal |
| `coordinator` | Manajemen alur kerja dan orkestrasi tugas |
| `hooks` | Custom hooks untuk state management |
| `ink` | Integrasi library Ink untuk UI terminal |
